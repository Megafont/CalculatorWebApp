Instructions

To run the React+NextJS version of the Calculator web app, open the root folder in VS Code and type the following into the terminal:

npm run dev

Full setup instructions can be found in this course, which is how I got started on this stuff during the P1 Discord bot project. As you can see on that page, you can install the required packages using the following command:

npm install react@latest react-dom@latest next@latest

This will install the latest versions of React, React-DOM, and NextJS. Remember, DOM stands for Document Object Model. These packages were too large to just commit into the GitHub repository, but you need to have them all installed in the project folder or the npm run dev command won't work.
